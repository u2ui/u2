# PointerObserver.js
Observe mouse and touches

## Usage

```js
new u2_PointerObserver(el).onmove = function(e){
    console.log(this.pos);
}
```

```html
<div id=el style="position:absolute;">
    StartObserving
</div>
```

## Install

```js
import * as module from "https://cdn.jsdelivr.net/gh/u2ui/u2@x.x.x/js/PointerObserver/PointerObserver.js"
```
