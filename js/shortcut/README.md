# shortcut.js
Easy keyboard shortcuts

## Usage

```js
listen('a+b+c', () => {
    alert('a+b+c pressed');
}, {target: testEl});
```

```html
<textarea id=testEl>write "abc" while holding the keys</textarea>
```
