# .u1-unstyle - class
Unstyle elements eg. buttons

Specificity: 0.1.0

## Ussage

```html
Button:
<button class=u1-unstyle>button</button>
<br>
<label>
    Input:
    <input class=u1-unstyle value="usgin u1-unstyle">
</label>
```

```css
.u1-unstyle {
}
```

## Install

```html
<link href="../../../unstyle.class@x.x.x/unstyle.min.css" rel=stylesheet>
```
