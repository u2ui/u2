# .u2-badge - class
Rendering badges

## Usage

```html
<span class=u2-badge>Badge</span>
```

## Install

```html
<link href="https://cdn.jsdelivr.net/gh/u2ui/u2@x.x.x/class/badge/badge.min.css" rel=stylesheet>
```
