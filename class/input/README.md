# .u1-input - class
custom inputs (beta)

## Ussage

```html
<label class=u1-input>
    <input value="test">
    <button type=button>save</button>
</label>
<br>
<br>
<label class=u1-input>
    <input value="1">
    <button type=button style="order:-1">-</button>
    <button type=button>+</button>
</label>
<label>
    Normal input:
    <input>
</label>
```

```css
.u1-input {
    border:2px solid;
}
.u1-input button {
    padding:.2em .5em;
    --line-height:1;
}
```

## Install

```html
<link href="../../../input.class@x.x.x/input.min.css" rel=stylesheet>
```
