# .u1-reset - class
A class to reset Styles (beta)

Also intended to reset own styles:

```css
button:hover {
    transform:scale(1.2);
}

.u1-reset button:hover {
    transform:none;
}
```

**in Progress!**

## Ussage

```html
<h1>Heading 1</h1>
<div class=u1-reset>
    <h1>Heading 1</h1>
</div>
```

```css
h1 {
    margin:0 2rem;
}
```

## Install

```html
<link href="../../../reset.class@x.x.x/reset.min.css" rel=stylesheet>
```
