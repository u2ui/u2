# .u2-reset - class
A class to reset Styles (beta)

Also intended to reset own styles:

```css
button:hover {
    transform:scale(1.2);
}

.u2-reset button:hover {
    transform:none;
}
```

**in Progress!**

## Usage

```html
<h1>Heading 1</h1>
<div class=u2-reset>
    <h1>Heading 1</h1>
</div>
```

```css
h1 {
    margin:0 2rem;
}
```

## Install

```html
<link href="https://cdn.jsdelivr.net/gh/u2ui/u2@main/class/reset/reset.css" rel=stylesheet>
```
