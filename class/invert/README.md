# .u1-invert - class
invert colors

## Ussage

```html
<h3>Normal H3</h3>
<div class=u1-invert style="padding:2rem">
    <h3>This box...</h3>
    <p>
        ...is inverted
        <a href="#">a link</a>
    </p>
    <button>button</button>
</div>
```

```css
.u1-invert {
}
```

## Install

```html
<link href="../../../invert.class@x.x.x/invert.min.css" rel=stylesheet>
```
