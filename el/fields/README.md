# &lt;u2-fields&gt; - element
Every content before a form element is considered as label

## Usage

```html
<u2-fields>
    Name<input type="text">
    Accept<input type="checkbox">
</u2-fields>
```

```css
u2-fields {
}
```

## Install

```html
<link href="https://cdn.jsdelivr.net/gh/u2ui/u2@x.x.x/el/fields/fields.min.css" rel=stylesheet>
<script src="https://cdn.jsdelivr.net/gh/u2ui/u2@x.x.x/el/fields/fields.min.js" type=module async></script>
```
