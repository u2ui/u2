# &lt;u2-textfit&gt; - element
Exactly fit text

## Usage

```html
<u2-textfit>
    <span contenteditable>Textfit</span>
</u2-textfit>
```

```css
u2-textfit {
    font-size: clamp(20px, var(--js-font-size), 400px);
}
```

## Install

```html
<link href="https://cdn.jsdelivr.net/gh/u2ui/u2@main/el/textfit/textfit.css" rel=stylesheet>
<script src="https://cdn.jsdelivr.net/gh/u2ui/u2@main/el/textfit/textfit.js" type=module async></script>
```
