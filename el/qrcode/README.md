# &lt;u1-qrcode&gt; - element
QR-Code element

## Ussage

```html
<u1-qrcode>Hello world</u1-qrcode>
```

```css
u1-qrcode {
    color:#555;
    max-width:10rem;
    margin:auto;
}
```

## Install

```html
<link href="../../../qrcode.el@x.x.x/qrcode.min.css" rel=stylesheet>
<script src="../../../qrcode.el@x.x.x/qrcode.min.js" type=module>
```
