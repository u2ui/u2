# &lt;u2-spot&gt; - element
Visual indicator tracking elements by selector

## Usage

```html
<u2-spot target=":scope > h6" active=":hover"></u2-spot>
<h5>Target:</h5>
<h6>By hover me</h6>
<h6>Or hover me</h6>
```

```css
u2-spot {
    transition-duration: 0.5s;
    background:var(--color-light);
}
```

## Install

```html
<link href="https://cdn.jsdelivr.net/gh/u2ui/u2@main/el/spot/spot.css" rel=stylesheet>
<script src="https://cdn.jsdelivr.net/gh/u2ui/u2@main/el/spot/spot.js" type=module async></script>
```
