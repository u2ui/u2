# &lt;u2-video&gt; - element
Video-element with controls and keyboard shortcuts.

## Usage

```html
<u2-video controls keyboard-shortcuts playback-speed
          src="https://interactive-examples.mdn.mozilla.net/media/cc0-videos/flower.mp4">
</u2-video>
```

```css
u2-video {
    font-size:13px;
    color:pink;
    border-radius: 1rem;
    box-shadow:0 0 1rem #0008;
    --gap:.5em;
}
```

## Install

```html
<script src="https://cdn.jsdelivr.net/gh/u2ui/u2@main/el/video/video.js" type=module async></script>
```
