# &lt;u2-carousel-nav&gt; - element
null

## Usage

```html
<u2-carousel-nav></u2-carousel-nav>
<u2-carousel>
    <div>Item 1</div>
    <div>Item 2</div>
    <div>Item 3</div>
</u2-carousel>
```

```css
u2-carousel-nav > * {
    transition:.3s;
}
u2-carousel-nav > .-active {
    transform:scale(1.2);
}
```

## Install

```html
<link href="https://cdn.jsdelivr.net/gh/u2ui/u2@x.x.x/el/carousel-nav/carousel-nav.min.css" rel=stylesheet>
```
