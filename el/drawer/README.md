# &lt;u2-drawer&gt; - element

## Usage

```html
<u2-drawer id="drawer">
    <div>Hello World</div>
</u2-drawer>
<button popovertarget="drawer" popovertargetaction="show-modal">open drawer</button>
```

```css
u2-drawer {
}
```

## Install

```html
<link href="https://cdn.jsdelivr.net/gh/u2ui/u2@x.x.x/el/drawer/drawer.min.css" rel=stylesheet>
<script src="https://cdn.jsdelivr.net/gh/u2ui/u2@x.x.x/el/drawer/drawer.min.js" type=module async></script>
```
