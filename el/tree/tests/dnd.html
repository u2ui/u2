Rules (via u2-tree-dragover veto): you cannot drop into locked folders (myaccess<2, dimmed); a content node (italic) accepts no page. Locked but filled folders still expand on hover (just slower) → so you can reach droppable sub-pages. Moving runs server-first (simulated, ~400 ms) via u2-tree-drop. You drag by the grip (via attr/draghandle) — so no ancestor is ever dragged along. Nodes with myaccess<2 have no grip → not draggable.
