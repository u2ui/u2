# [u2-behavior] - attribute
behavior before action

## Usage

```html
<button u2-behavior=share>behavior=share</button>
```

## Install

```html
<script src="https://cdn.jsdelivr.net/gh/u2ui/u2@x.x.x/attr/behavior/behavior.min.js" type=module async></script>
```
