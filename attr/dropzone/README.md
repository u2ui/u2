

## Usage

```html
<div u2-dropzone="p">
    <p draggable="true">item 1</p>
    <p draggable="true">item 2</p>
</div>
<p draggable="true">item 3</p>
```

## Install

```html
<script src="https://cdn.jsdelivr.net/gh/u2ui/u2@x.x.x/attr/dropzone/dropzone.min.js" type=module async></script>
```
