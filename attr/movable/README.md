# [u2-movable] - attribute
Make an element movable. Optionally define a [u2-movable-handler] element.

## Usage

```html
<div u2-movable>
    <div u2-movable-handler>movable handle</div>
</div>
```

## Install

```html
<script src="https://cdn.jsdelivr.net/gh/u2ui/u2@x.x.x/attr/movable/movable.js" type=module async></script>
```
