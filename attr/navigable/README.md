# [u1-navigable] - attribute
Make elements navigatable

## Usage

```html
<a href="#navigable">open dialog</a>
<dialog u1-navigable id=navigable>
    go back using esc
</dialog>
```

## Install

```html
<script src="../../../navigable.attr@x.x.x/navigable.min.js" type=module></script>
```
